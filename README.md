# Research Paper Notes

A simple Markdown repository for collecting paper-reading notes and blog-style explanations.

## Current contents

- `papers/README.md` — index of papers already read, with paper/project/code links.
- `blog_hieu_transformer_tu_goc.md` — Vietnamese blog note for **Attention Is All You Need**.
- `paper_note_attention_is_all_you_need.md` — mirrored paper note content for **Attention Is All You Need**.
- `notes/read-paper-template.md` — reusable template for future paper notes.
- `papers/` — place local paper references or tracked paper indexes here.
- `notes/` — place future Markdown notes here.

## Workflow

1. Read a paper.
2. Create or copy a Markdown note from `notes/read-paper-template.md`.
3. Add the paper to `papers/README.md`.
4. If needed, paste the finished Markdown into Notion manually.

## Repository rules

- Keep notes and indexes in Markdown.
- Do not commit secrets.
- Do not commit large PDF files unless you intentionally want them in Git.
